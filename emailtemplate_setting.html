Image Filters

To include images from your store, use the base64_img filter - for example:

<img src="{{ '//cdn.shopify.com/s/files/1/0031/0882/products/powerboat_large_large.jpg' | base64_img }}">

Map Filters

You can include a Google Maps image for the location of an item using the gmaps_img filter. This filter accepts parameters for the zoom level and size (width x height).

<img src="{{ item.location | gmaps_img: 'zoom:16,size:225x150' }}">

Note: the location assigned to the booking item comes from the list of locations assigned to the product.
